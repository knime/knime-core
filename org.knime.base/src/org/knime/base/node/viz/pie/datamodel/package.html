This package contains the pie chart data model classes.

A pie chart in general consists of a {@link org.knime.base.node.viz.pie.datamodel.PieVizModel} which contains the visualization specific information like pie size, label policy, and layout flags. It also holds the {@link org.knime.base.node.viz.pie.datamodel.PieDataModel} which consists of the {@link org.knime.base.node.viz.pie.datamodel.PieSectionDataModel}s. Each pie sections represents a value of the selected pie column. The sections contain one or more {@link org.knime.base.node.viz.pie.datamodel.PieSubSectionDataModel}s. Each sub section represents a color defined on a column using the color manager.
